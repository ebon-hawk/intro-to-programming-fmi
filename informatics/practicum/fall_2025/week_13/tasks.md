# Рекурсия

### Задача 1 → Сума на числата от 1 до `N`

Напишете **рекурсивна функция**, която приема цяло положително число `N` и връща сумата на всички числа от `1` до `N`.

```
Вход:
N = 5

Изход:
15
```

### Задача 2 → Степенуване

Като използвате **рекурсивната дефиниция на степенуването**, напишете функция, която по дадени цели положителни числа `p` и `q` намира стойността на `p ^ q`.

```
Вход:
p = 2, q = 5

Изход:
32
```

### Задача 3 → Сума от цифрите на число

Напишете **рекурсивна функция**, която приема цяло число `x` и връща сумата от цифрите му.

```
Вход:
x = 1234

Изход:
10
```

### Задача 4 → Проверка за съдържание на цифра

Напишете **рекурсивна функция**, която приема две цели числа `n` и `k` и проверява дали в десетичния запис на `n` се съдържа цифрата `k`.

```
Вход:
n = 57234, k = 3

Изход:
true
```

### Задача 5 → Монотонно растяща редица

Напишете **рекурсивна програма**, която проверява дали редицата `a₀, a₁, ..., aₙ₋₁` е **монотонно растяща**.

Редица е монотонно растяща, ако:

```
a₀ <= a₁ <= a₂ <= ... <= aₙ₋₁
```

### Задача 6 → Вмъкване в сортиран масив

Напишете **рекурсивна функция**, която приема:

- Сортиран масив от цели числа
- Цяло число `x`

... и добавя числото `x` на правилната позиция, така че масивът да остане сортиран.

### Задача 7 → Палиндром чрез премахване на цифри

Напишете **рекурсивна функция**, която приема цяло число `n` и проверява дали чрез **няколко премахвания на първата или последната цифра** може да се получи **палиндром**.

```
Вход:
n = 12321

Изход:
true
```

### Задача 8 → Области в квадратна мрежа (*Flood Fill*)

Дадена е **квадратна мрежа от клетки**, всяка от които е празна или запълнена. Запълнените клетки, които са свързани **хоризонтално, вертикално или диагонално**, образуват **област**.

Да се напише **рекурсивна програма**, която:

- Намира **броя на областите**
- Намира **размера (брой клетки)** на всяка област

### Задача 9 → Комбинаторни конфигурации с числа

Реализирайте **рекурсивни функции**, които генерират и извеждат на екрана различни **комбинаторни конфигурации** като последователности от числа.

Да се реализират:

- Комбинации без повторение
- Комбинации с повторение
- Вариации без повторение
- Вариации с повторение
- Пермутации без повторение
- Пермутации с повторение
- Всички подмножества на множество

### Задача 10 → Комбинаторни конфигурации с произволни елементи

Променете реализациите от предходната задача така, че да **не работят с числата от 1 до `N`**, а с **подадено множество от елементи**, например:

- Символни низове
- Символи

```
Вход:
{ "apple", "banana", "cherry" }

Изход:
apple banana
apple cherry
banana cherry
```
