# Символни низове. Стандартни функции за работа със символни низове

### Задача 1 → Имплементация на `atoi`

Напишете функция, която преобразува символен низ, съдържащ представяне на цяло число, в числов тип `int`.

```
Вход:
"12345"

Изход:
12345
```

### Задача 3 → Имплементация на `searchInText`

Напишете функция, която търси дали един низ се съдържа в друг.

```
Вход:
text = "Hello world!"
pattern = "world"

Изход:
true
```

### Задача 4 → Замяна на низ в текст (in-place)

Напишете функция, която приема три стринга: `text`, `where` и `what`. Трябва да замени във `text` всяко срещане на `where` с `what` **без да използва допълнителна памет**.

```
Вход:
text = "I am the best of the best."
where = "best"
what = "worst"

Изход:
"I am the worst of the worst."
```

### Задача 6 → Най-често срещана дума

```
Вход:

```
